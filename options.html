HolodexPlusExtension

進階設定

監控頻道清單

    輸入頻道名稱，例如：Koseki Bijou 或 Koseki Bijou Ch. hololive-EN。每個頻道可點擊 ⚙ 設定個別選項。

    監控標籤（topic_id）

      輸入 API 回傳的 topic_id 字段，符合該標籤的所有直播都會被自動開啟

      topic／關鍵字的 org 限制

      只有 channel.org 與此值相符（不分大小寫）的直播，才會套用 topic_id 和標題關鍵字規則。留空代表不限 org。預設：Hololive

      監控標題關鍵字（title）

      標題包含此關鍵字（不分大小寫）的所有直播都會被自動開啟

      時間設定

      在排程開播前幾分鐘自動開啟分頁
      已開播後距今幾分鐘內若分頁被關閉，自動重開
